# how-to

_📚 Doing all sorts of things, the DataMade way_

## What's this?

Here at DataMade, we do a lot of computer programming. In the spirit of [better living through documentation](https://datamade.us/blog/better-living-through-documentation), we're preserving guides to how we do that here.

We are currently in the process of updating our documentation and practices around documentation, so the contents of this repo have been archived in the archive folder, to be brought back into the root directory as they are found useful. 

The previous iteration of the repo is preserved in the `archive` branch for reference.
